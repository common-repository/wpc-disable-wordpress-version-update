WPC Disable Wordpress Version Updates removes the Wordpress update feature.

== Description ==

WPC Disable Wordpress Version Updates removes the Wordpress update feature. It will get rid of that annoying update warning header that appears if you need to update. Lots of users don’t like to update their sites so this is perfect for them, or any site admin that don’t want their clients seeing update info they shouldn’t.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Unzip and upload to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to the plugin options page link in the admin menu and set up.
